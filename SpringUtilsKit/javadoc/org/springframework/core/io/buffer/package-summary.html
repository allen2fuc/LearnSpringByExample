Package org.springframework.core.io.buffer

Generic abstraction for working with byte buffer implementations.

See: Description

  • Interface Summary 
    Interface Description
    TouchableDataBufferExtension of DataBuffer that allows for buffers that can be given hints for debugging purposes.
    CloseableDataBufferExtension of DataBuffer that allows for buffers that can be used in a try-with-resources statement.
    MatcherContract to find delimiter(s) against one or more data buffers that can be passed one at a time to the match method.
    DataBufferFactoryA factory for DataBuffers, allowing for allocation and wrapping of data buffers.
    PooledDataBufferExtension of DataBuffer that allows for buffers that share a memory pool.
    ByteBufferIteratorA dedicated iterator type that ensures the lifecycle of iterated ByteBuffer elements.
    DataBufferBasic abstraction over byte buffers.

    NettyDataBufferFactoryImplementation of the DataBufferFactory interface based on a Netty 4 ByteBufAllocator.

    DefaultDataBufferFactoryDefault implementation of the DataBufferFactory interface.
    LimitedDataBufferListCustom List to collect data buffers with and enforce a limit on the total number of bytes buffered.
    DataBufferUtilsUtility class for working with DataBuffers.
    NettyDataBufferImplementation of the DataBuffer interface that wraps a Netty 4 ByteBuf.
    Netty5DataBufferFactoryImplementation of the DataBufferFactory interface based on a Netty 5 BufferAllocator.
