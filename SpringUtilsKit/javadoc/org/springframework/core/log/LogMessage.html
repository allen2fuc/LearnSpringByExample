Package org.springframework.core.log

Class LogMessage

  • All Implemented Interfaces:
    java.lang.CharSequence 
    
public abstract class LogMessage
 implements CharSequence

    A simple log message type for use with Commons Logging, allowing for convenient lazy resolution of a given Supplier instance (typically bound to a lambda expression) or a printf-style format string (format) in its toString.
