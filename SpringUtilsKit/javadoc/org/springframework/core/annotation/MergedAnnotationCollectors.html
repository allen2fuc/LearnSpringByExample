      • toAnnotationSet

         static <A extends Annotation> Collector<MergedAnnotation<A>, out Object, Set<A>> toAnnotationSet()

        Create a new Collector that accumulates merged annotations to a LinkedHashSet containing synthesized versions.

        The collector returned by this method is effectively equivalent to Collectors.mapping(MergedAnnotation::synthesize, Collectors.toCollection(LinkedHashSet::new)) but avoids the creation of a composite collector.
