Package org.springframework.core.task

Class VirtualThreadTaskExecutor

  • All Implemented Interfaces:
    java.util.concurrent.Executor , org.springframework.core.task.AsyncTaskExecutor , org.springframework.core.task.TaskExecutor 
    
public class VirtualThreadTaskExecutor
 implements AsyncTaskExecutor

    A TaskExecutor implementation based on virtual threads in JDK 21+. The only configuration option is a thread name prefix.

    For additional features such as concurrency limiting or task decoration, consider using setVirtualThreads instead.
