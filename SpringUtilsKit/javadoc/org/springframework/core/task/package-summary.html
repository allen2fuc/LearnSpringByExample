Package org.springframework.core.task

This package defines Spring's core TaskExecutor abstraction, and provides SyncTaskExecutor and SimpleAsyncTaskExecutor implementations.

See: Description

  • Interface Summary 
    Interface Description
    TaskDecoratorA callback interface for a decorator to be applied to any Runnable about to be executed.
    AsyncListenableTaskExecutorExtension of the AsyncTaskExecutor interface, adding the capability to submit tasks for ListenableFutures.
    AsyncTaskExecutorExtended interface for asynchronous TaskExecutor implementations, offering support for java.util.concurrent.Callable.
    TaskExecutorSimple task executor interface that abstracts the execution of a Runnable.
  • Class Summary 
    Class Description
    TaskTimeoutExceptionException thrown when a AsyncTaskExecutor rejects to accept a given task for execution because of the specified timeout.
    TaskRejectedExceptionException thrown when a TaskExecutor rejects to accept a given task for execution.
    SyncTaskExecutorTaskExecutor implementation that executes each task synchronously in the calling thread.
    SimpleAsyncTaskExecutorTaskExecutor implementation that fires up a new Thread for each task, executing it asynchronously.
    VirtualThreadTaskExecutorA TaskExecutor implementation based on virtual threads in JDK 21+.
