      • visit

         void visit(int version, int access, String name, String signature, String superName, Array<String> interfaces)

        Visits the header of the class.

        Parameters:
        version - the class version.
        access - the class's access flags (see Opcodes).
        name - the internal name of the class (see getInternalName).
        signature - the signature of this class.
        superName - the internal of name of the super class (see getInternalName).
        interfaces - the internal names of the class's interfaces (see getInternalName).

      • visitEnd

         void visitEnd()

      • visitMethod

         MethodVisitor visitMethod(int access, String name, String desc, String signature, Array<String> exceptions)

        Visits a method of the class. This method must return a new MethodVisitor instance (or null) each time it is called, i.e., it should not return a previously returned visitor.

        Parameters:
        access - the method's access flags (see Opcodes).
        name - the method's name.
        signature - the method's signature.
        exceptions - the internal names of the method's exception classes (see getInternalName).
